# List View Behaviors (GEN::LST)

Cross-cutting interaction patterns for entity list views. These behaviors are implemented by `createEntityDataGrid` and inherited by all entity grids (Items, Suppliers, etc.).

## Implementation

All list view behaviors are built into the `createEntityDataGrid` factory organism. Entity-specific grids (ItemGrid, supplier grid) inherit these behaviors automatically via the factory config. See the [Entity Data Grid docs](?path=/docs/components-canary-organisms-shared-entity-data-grid--docs) for the component API.
