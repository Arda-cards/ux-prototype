# Business Affiliates: Edit Supplier

<a href={docsUrl('/product/use-cases/reference-data/suppliers/business-affiliates/#refba0004----edit-business-affiliate')}
   target="_blank" rel="noopener">
  View full specification &#8594;
</a>

Edit an existing supplier from the detail drawer. The flow: click a row to open the
drawer in read-only view mode &#8594; click Edit to switch to edit mode &#8594;
modify fields &#8594; click Save to send the PUT request &#8594; receive a success toast
&#8594; drawer returns to view mode showing updated values.
