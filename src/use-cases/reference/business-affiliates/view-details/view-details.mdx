# Business Affiliates: View Details

The Supplier Details Panel opens as a side drawer when a row is clicked in the suppliers list.
It shows all affiliate fields organized in collapsible sections: Identity, Contact, Address,
Legal, and Notes.

Each section expands or collapses independently. Sections with data default to expanded;
sections with no data default to collapsed and show an empty-state message when expanded.

## Shared Drawer Component

The `SupplierDrawer` component (`_shared/supplier-drawer.tsx`) supports three modes:

- **view** — Read-only field display with Edit and Delete action buttons (this story)
- **create** — Blank form with Save and Cancel buttons (BA::0003::0001)
- **edit** — Pre-populated editable form with Save and Cancel buttons (BA::0004::0001)

Create mode (BA::0003::0001) and Edit mode (BA::0004::0001) reuse this same drawer component
with different `mode` props.
