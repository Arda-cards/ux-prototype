# Items Reference Use Cases

Use cases for the Items reference data domain &#8212; browsing, viewing, editing, and managing inventory items.

Items are the core reference data entity in Arda. Each item represents a physical inventory product with properties like name, SKU, classification, supplier, and cost information.
