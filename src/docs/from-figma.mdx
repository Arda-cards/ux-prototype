## Page Layout

### Overall Structure

Every authenticated page follows this layout:

```
+--------------------------------------------------+
| Sidebar (fixed)  |        Content Area            |
| w-[240px] or     | +----------------------------+ |
| w-[56px]         | | Page Header (breadcrumb +   | |
| collapsed        | | title + actions)            | |
|                  | +----------------------------+ |
| Dark bg #0A0A0A  | | Tab Navigation (optional)   | |
|                  | +----------------------------+ |
| - Logo           | |                            | |
| - Nav items      | |  Page Content              | |
| - User footer    | |  (tables, cards, forms...) | |
|                  | |                            | |
+------------------+----------------------------+---+
```

### Sidebar

- **Width:** 240px expanded, 56px collapsed
- **Background:** #0A0A0A with subtle gradient overlay
- **Position:** Fixed, full viewport height
- **Nav item default:** `text-white/70 hover:bg-white/10`
- **Nav item active:** `bg-white/10 text-white` with `border-l-2 border-[#FC5A29]`

### Page Header

Breadcrumb (`text-sm text-muted-foreground`) + page title (`text-xl font-extrabold`) + right-aligned action buttons.

### Tab Navigation

Horizontal tabs below the page header. Active tab has a primary-orange bottom border with `font-semibold`.

### Side Drawers

Slide-in panels from the right (400&#8211;600px wide) with header, close button, and backdrop overlay.
