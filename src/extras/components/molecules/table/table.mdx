# ArdaTable

A composite data table built from six sub-components. Renders inventory data with clickable rows, active-row highlighting, and status badges.

## Configuration Lifecycle

### ArdaTableRow — Static vs Runtime

The only sub-component with a meaningful lifecycle split:

- **Static:** `className` — Additional CSS classes set at composition time.
- **Runtime:** `onClick` — Row selection handler. `active` — Whether the row is in the selected/highlighted state.

All other sub-components accept only `children` and an optional `className` — they are purely structural.

## Accessibility

- Uses semantic HTML table elements (`table`, `thead`, `tbody`, `tr`, `th`, `td`).
- Clickable rows have `cursor: pointer` but no `role="button"` — consider adding `role="row"` with `aria-selected` for accessible selection patterns.
- Header cells use uppercase text styling. Screen readers will announce the original text content regardless of CSS text-transform.

## Related Components

- **ArdaBadge** — Commonly used inside table cells for status columns.
- **ArdaItemCard** — Alternative card-based item display.
