# Date Cell

Grid cell component for displaying and editing date values in AG Grid. Parses YYYY-MM-DD as local date to avoid timezone shifts.

## Modes

### Display Mode

Read-only date renderer. Shows &#8212; for empty/undefined values. Formats ISO date strings in local timezone.

<Canvas of={DateStories.Display} />

### Edit Mode

AG Grid cell editor with date input control, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={DateStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={DateStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={DateStories.EditableOverride} />
