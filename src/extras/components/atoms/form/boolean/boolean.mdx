# Boolean Field

Form field component for displaying and editing boolean values with multiple display formats.

## Modes

### Display Mode

Read-only boolean display. **Checkbox format** shows check/X icons. **Yes-No format** shows "Yes", "No", or "&#8212;" text.

<Canvas of={BooleanStories.Display} />

### Edit Mode

Interactive boolean input with format-specific controls. **Checkbox format** uses standard checkbox input. **Yes-No format** uses toggle buttons.

<Canvas of={BooleanStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={BooleanStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={BooleanStories.EditableOverride} />
