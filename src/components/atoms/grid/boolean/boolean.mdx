# Boolean Cell

Grid cell component for displaying and editing boolean values in AG Grid. Supports two display formats: checkbox icons and yes-no text.

## Modes

### Display Mode

Read-only boolean renderer. Shows &#8212; for empty/undefined values. **Checkbox format** shows check/X icons. **Yes-No format** shows "Yes", "No", or &#8212; text.

<Canvas of={BooleanStories.Display} />

### Edit Mode

AG Grid cell editor with checkbox or yes-no toggle, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={BooleanStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={BooleanStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={BooleanStories.EditableOverride} />
