# Image Cell

Grid cell component for displaying and editing image URLs in AG Grid. Displays images from URLs with configurable sizing.

## Modes

### Display Mode

Read-only image renderer. Shows placeholder icon for empty/undefined values or broken image URLs. Displays image thumbnail with specified dimensions.

<Canvas of={ImageStories.Display} />

### Edit Mode

AG Grid cell editor with URL input control, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={ImageStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={ImageStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={ImageStories.EditableOverride} />
