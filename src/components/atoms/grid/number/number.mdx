# Number Cell

Grid cell component for displaying and editing numeric values in AG Grid. Configurable decimal precision with right-aligned display.

## Modes

### Display Mode

Read-only number renderer. Shows &#8212; for empty/undefined values. Formats numbers with specified decimal precision.

<Canvas of={NumberStories.Display} />

### Edit Mode

AG Grid cell editor with numeric input, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={NumberStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={NumberStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={NumberStories.EditableOverride} />
