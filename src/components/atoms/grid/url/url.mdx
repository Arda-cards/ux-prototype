# URL Cell

Grid cell component for displaying and editing URL values in AG Grid. Supports two display formats: text link and styled button.

## Modes

### Display Mode

Read-only URL renderer. Shows &#8212; for empty/undefined values. Displays either as clickable link text or as a styled button with external-link icon.

<Canvas of={UrlStories.Display} />

### Edit Mode

AG Grid cell editor with URL input control, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={UrlStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={UrlStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={UrlStories.EditableOverride} />
