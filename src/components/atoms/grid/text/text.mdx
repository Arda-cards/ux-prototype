# Text Cell

Grid cell component for displaying and editing text values in AG Grid.

## Modes

### Display Mode

Read-only text renderer. Shows &#8212; for empty/undefined values. Supports truncation via `maxLength`.

<Canvas of={TextStories.Display} />

### Edit Mode

AG Grid cell editor with auto-focus, Enter to commit, Escape to cancel. Exposes `getValue()` via ref for AG Grid integration.

<Canvas of={TextStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={TextStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={TextStories.EditableOverride} />
