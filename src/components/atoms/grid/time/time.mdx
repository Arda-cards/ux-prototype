# Time Cell

Grid cell component for displaying and editing time values in AG Grid. Formats HH:mm to 12-hour AM/PM format.

## Modes

### Display Mode

Read-only time renderer. Shows &#8212; for empty/undefined values. Converts 24-hour time to 12-hour format with AM/PM.

<Canvas of={TimeStories.Display} />

### Edit Mode

AG Grid cell editor with time input control, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={TimeStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={TimeStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={TimeStories.EditableOverride} />
