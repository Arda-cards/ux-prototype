# Date Time Cell

Grid cell component for displaying and editing combined date and time values in AG Grid.

## Modes

### Display Mode

Read-only date-time renderer. Shows &#8212; for empty/undefined values. Formats ISO datetime strings using `Intl.DateTimeFormat`.

<Canvas of={DateTimeStories.Display} />

### Edit Mode

AG Grid cell editor with datetime-local input control, auto-focus, Enter to commit, Escape to cancel.

<Canvas of={DateTimeStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={DateTimeStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={DateTimeStories.EditableOverride} />
