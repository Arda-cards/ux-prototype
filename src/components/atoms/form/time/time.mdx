# Time Field

Form field component for displaying and editing time values with 12-hour AM/PM formatting.

## Modes

### Display Mode

Read-only time formatted as 12-hour AM/PM (e.g., "2:30 PM"). Shows &#8212; for empty values. Accepts time strings in HH:mm or HH:mm:ss format.

<Canvas of={TimeStories.Display} />

### Edit Mode

Native time picker input. Uses browser's native time selection UI with keyboard and mouse support.

<Canvas of={TimeStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={TimeStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={TimeStories.EditableOverride} />
