# Date Time Field

Form field components for displaying and editing combined date and time values.

## Variants

### Display

Read-only datetime formatted as "MM/DD/YYYY h:mm AM/PM" (e.g., "03/15/2024 2:30 PM"). Shows `&#8212;` for empty values. Accepts ISO datetime strings.

<Canvas of={DateTimeStories.Display} />

### Editor

Native datetime-local picker input (`<input type="datetime-local">`). Uses browser's native datetime selection UI with combined date and time controls.

<Canvas of={DateTimeStories.Editor} />

### Interactive

Starts in display mode. Double-click to switch to editing with native datetime picker. Enter/blur commits, Escape cancels.

<Canvas of={DateTimeStories.Interactive} />

## Configuration

### Static Configuration (Design Time)

- **placeholder** &#8212; Placeholder text for the input

### Runtime Configuration

- **value** &#8212; ISO datetime string (YYYY-MM-DDTHH:mm:ssZ)
- **onChange** &#8212; Called when datetime value changes
- **onComplete** &#8212; Called on Enter or blur
- **onCancel** &#8212; Called on Escape
- **disabled** &#8212; Disables editing
- **autoFocus** &#8212; Focus the input on mount
