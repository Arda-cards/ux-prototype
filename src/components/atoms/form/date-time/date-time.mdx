# Date Time Field

Form field component for displaying and editing combined date and time values.

## Modes

### Display Mode

Read-only datetime formatted as "MM/DD/YYYY h:mm AM/PM". Shows &#8212; for empty values. Accepts ISO datetime strings.

<Canvas of={DateTimeStories.Display} />

### Edit Mode

Native datetime-local picker input. Uses browser's native datetime selection UI with combined date and time controls.

<Canvas of={DateTimeStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={DateTimeStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={DateTimeStories.EditableOverride} />
