# URL Field

Form field component for displaying and editing URL values with multiple display formats.

## Modes

### Display Mode

Read-only URL display with two format options. **Link format** shows clickable blue hyperlink with truncation. **Button format** shows styled button with customizable label and external link icon. Both open in new tab by default. Shows &#8212; for empty values.

<Canvas of={UrlStories.Display} />

### Edit Mode

URL input with browser validation. Uses native URL validation pattern. Standard text input styling with focus ring.

<Canvas of={UrlStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={UrlStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={UrlStories.EditableOverride} />
