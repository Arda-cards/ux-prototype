# Text Field

Form field component for displaying and editing text values with optional truncation.

## Modes

### Display Mode

Read-only text with form field styling. Shows &#8212; for empty values. Supports `maxLength` truncation with ellipsis.

<Canvas of={TextStories.Display} />

### Edit Mode

Standard text input with border, focus ring, and keyboard handlers (Enter to complete, Escape to cancel).

<Canvas of={TextStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={TextStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={TextStories.EditableOverride} />
