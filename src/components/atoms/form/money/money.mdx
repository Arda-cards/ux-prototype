# Money Field

Composite form field for monetary values&#8212;a number input (amount) paired with a currency selector.

## Variants

### Display

Read-only formatted money value. Shows currency symbol via `Intl.NumberFormat` plus the currency code. Shows `&#8212;` for empty values.

<Canvas of={MoneyStories.Display} />

### Editor

Number input for the amount alongside a `select` dropdown for the currency code, laid out in a flex row.

<Canvas of={MoneyStories.Editor} />

### Interactive

Mode-driven rendering: display, edit, or error. Respects `editable` override.

<Canvas of={MoneyStories.InteractiveEdit} />

## Configuration

### Static Configuration (Design Time)

- **currencyOptions** &#8212; Record mapping currency codes to display names
- **precision** &#8212; Number of decimal places (default: 2)
- **placeholder** &#8212; Placeholder text for the amount input

### Runtime Configuration

- **value** &#8212; Object with `amount` (number) and `currency` (string)
- **onChange** &#8212; Called on every change with original and current values
- **onComplete** &#8212; Called on Enter or blur
- **onCancel** &#8212; Called on Escape
