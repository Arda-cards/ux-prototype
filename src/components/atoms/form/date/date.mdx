# Date Field

Form field component for displaying and editing date values with optional timezone support.

## Modes

### Display Mode

Read-only date formatted as MM/DD/YYYY. Shows &#8212; for empty values. Accepts ISO date strings or ISO datetime strings (extracts date portion). Supports optional IANA timezone for conversion.

<Canvas of={DateStories.Display} />

### Edit Mode

Native date picker input. Uses browser's native date selection UI with keyboard and mouse support.

<Canvas of={DateStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={DateStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={DateStories.EditableOverride} />
