# Image Field

Form field component for displaying and editing image URLs with preview.

## Modes

### Display Mode

Read-only image preview from URL. Shows image with configurable max dimensions and object-fit cover. Displays URL below the image. Shows placeholder icon with "No image" text for empty or broken URLs.

<Canvas of={ImageStories.Display} />

### Edit Mode

URL input for image with live preview. Validates URL by attempting to load the image. Shows preview below input when URL is valid.

<Canvas of={ImageStories.Editor} />

### Error Mode

Edit mode with error styling and error messages displayed below the input.

<Canvas of={ImageStories.InteractiveError} />

### Editable Override

When `editable` is set to `false`, the component renders in display mode regardless of the `mode` prop.

<Canvas of={ImageStories.EditableOverride} />
