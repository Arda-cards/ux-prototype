# Duration Field

Composite form field for time durations&#8212;a number input (value) paired with a time unit selector.

## Variants

### Display

Read-only formatted duration value showing the numeric value and unit code. Shows `&#8212;` for empty values.

<Canvas of={DurationStories.Display} />

### Editor

Number input for the value alongside a `select` dropdown for the time unit, laid out in a flex row.

<Canvas of={DurationStories.Editor} />

### Interactive

Mode-driven rendering: display, edit, or error. Respects `editable` override.

<Canvas of={DurationStories.InteractiveEdit} />

## Configuration

### Static Configuration (Design Time)

- **unitOptions** &#8212; Record mapping unit codes to display names
- **precision** &#8212; Number of decimal places (default: 0)
- **placeholder** &#8212; Placeholder text for the value input

### Runtime Configuration

- **value** &#8212; Object with `value` (number) and `unit` (string)
- **onChange** &#8212; Called on every change with original and current values
- **onComplete** &#8212; Called on Enter or blur
- **onCancel** &#8212; Called on Escape
