# ImageInspectorOverlay

Full-size image modal overlay with an optional Edit button. Built on the Dialog atom in
controlled mode &#8212; the caller manages open/closed state via `open` and `onClose`.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/image-inspector-overlay`

---

## Playground

Click the button to open the overlay. Use the Controls panel to adjust `imageUrl` and `open`.

<Canvas of={Stories.Playground} />

---

## Inspector to Edit

When `onEdit` is provided, an Edit button appears in the footer. Clicking it calls
`onEdit` and then closes the overlay.

<Canvas of={Stories.WithEditButton} />

---

## Dismiss Methods

Three ways to close the overlay:

<ul>
  <li>Press <strong>Escape</strong></li>
  <li>Click the <strong>backdrop</strong> outside the image panel</li>
  <li>Click the <strong>X button</strong> in the top-right corner</li>
</ul>

<Canvas of={Stories.DismissMethods} />
