## Purpose

`ItemDetailsCardPreview` renders a centered carousel strip. All card elements are rendered in the DOM; a CSS `translateX` shift centers the active card. Peek opacity (30%) on adjacent cards hints at scrollability.

Navigation: prev/next arrow buttons, click-on-card, or swipe (touch). Single-card mode hides both arrows.
