# ArdaFieldList

Data-driven list of read-only label/value fields with dividers.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/field-list/field-list`

---

## Default

<Canvas of={FieldListStories.Default} />

---

## Purpose

`ArdaFieldList` renders a vertical list of label/value pairs using `ReadOnlyField` atoms
separated by thin dividers. Used in the `ArdaItemDetails` organism to display entity
metadata. Pass a `fields[]` array; for custom value rendering (links, formatted numbers,
badges) use the `children` property on each `FieldDef`.
