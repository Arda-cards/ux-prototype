# Sidebar Molecules

Sub-components that compose `ArdaSidebar`. These molecules are not used standalone &#8212;
they are always used inside an `ArdaSidebar` organism.

**Layer**: Molecule &#8212; Canary

---

## Sub-components

<table>
  <thead>
    <tr>
      <th>Component</th>
      <th>Import</th>
      <th>Purpose</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>ArdaSidebarHeader</code></td>
      <td><code>@/components/canary/molecules/sidebar/sidebar-header</code></td>
      <td>Brand logo + team name in the sidebar header. Pass <code>teams</code> for a team switcher dropdown.</td>
    </tr>
    <tr>
      <td><code>SidebarNav</code></td>
      <td><code>@/components/canary/molecules/sidebar/sidebar-nav</code></td>
      <td>Scrollable navigation area. Wraps children in a <code>nav</code> landmark.</td>
    </tr>
    <tr>
      <td><code>SidebarNavItem</code></td>
      <td><code>@/components/canary/molecules/sidebar/sidebar-nav-item</code></td>
      <td>Nav button with icon, label, optional badge, and active state. Badges collapse to dots in icon-only mode.</td>
    </tr>
    <tr>
      <td><code>SidebarNavGroup</code></td>
      <td><code>@/components/canary/molecules/sidebar/sidebar-nav-group</code></td>
      <td>Collapsible section with chevron. Auto-expands when a child item has <code>active=&#123;true&#125;</code>.</td>
    </tr>
    <tr>
      <td><code>SidebarUserMenu</code></td>
      <td><code>@/components/canary/molecules/sidebar/sidebar-user-menu</code></td>
      <td>Footer dropdown with avatar, user name, role, and configurable actions array.</td>
    </tr>
  </tbody>
</table>
