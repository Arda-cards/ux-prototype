# ImageFormField

Form field renderer for entity image fields. Wraps an `ImageDisplay` thumbnail inside an
`ImageHoverPreview` trigger and surfaces hover action icons for inspecting, editing, and
removing the image.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/form/image`

---

## Overview

`ImageFormField` manages the full read &#8212; and remove &#8212; surface for a single image property on
an entity form. Upload and crop are delegated to a parent dialog (opened via the Pencil action).

Key behaviours:

- Thumbnail renders at `w-24 h-24` by default; the container clips with `rounded-lg`.
- Hover overlay fades in via `opacity-0 group-hover:opacity-100 transition-opacity`.
- Eye and Trash actions are only shown when an image URL is present.
- Remove confirmation is an `AlertDialog` (no accidental deletes).
- Double-clicking the thumbnail is equivalent to clicking the Pencil icon.
- `disabled` prop sets `opacity-50 pointer-events-none` on the root.
