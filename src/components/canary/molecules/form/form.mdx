# Form Field Molecules

Form-specific field renderers that mirror the `atoms/grid/` cell-type pattern
for use inside entity create/edit forms.

**Layer**: Molecules &#8594; Form

---

## Pattern

Each form field type gets its own subdirectory under `form/`:
`form/<type>/`. The naming convention is `{Type}FormField`, paralleling the
`{Type}CellDisplay` / `{Type}CellEditor` naming in the grid layer.

This creates a uniform `{Type}{Layer}{Role}` naming pattern across the library:
