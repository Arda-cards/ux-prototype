# AddressFieldset

Compact structured postal-address form (street / unit / city / state / ZIP /
country) over the `PostalAddress` value object — the shared editing surface
for anywhere an address is captured (PO deliver-to, vendor details, company
settings). Controlled and null-tolerant: an all-blank form reports `null`, so
callers can distinguish "no address" from a partial one.
