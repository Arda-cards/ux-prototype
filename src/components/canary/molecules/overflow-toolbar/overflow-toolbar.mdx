# OverflowToolbar

Responsive container that collapses children into a dropdown when they no longer fit.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/overflow-toolbar/overflow-toolbar`

---

## Default

<Canvas of={OverflowToolbarStories.Default} />

## Purpose

`OverflowToolbar` renders `children` inline. When the container is too narrow to show
all children, items that overflow collapse into a `MoreHorizontal` dropdown. It uses
`ResizeObserver` to adapt to actual container width &#8212; no manual breakpoints needed.

Use this when you compose your own child buttons/dropdowns and want automatic overflow.
For a data-driven action list with fixed structure, use `ArdaActionToolbar` instead.

Each child should have a `data-overflow-label` attribute so the overflow menu can label
the item. If omitted, the child's `aria-label` or a numbered fallback is used.
