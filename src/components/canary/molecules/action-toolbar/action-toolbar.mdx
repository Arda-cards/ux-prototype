# ArdaActionToolbar

Responsive action bar that promotes items from an overflow dropdown as container width allows.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/action-toolbar/action-toolbar`

---

## Default

<Canvas of={ActionToolbarStories.Default} />

---

## Purpose

`ArdaActionToolbar` renders a list of `ToolbarAction` items as icon+label buttons when
space allows, collapsing overflow items into a `MoreHorizontal` dropdown. It uses
`ResizeObserver` to measure available container width at runtime &#8212; no breakpoints
needed.

Use this when you have a known action set and want automatic overflow behavior.
For dynamically composed children use `OverflowToolbar` instead.
