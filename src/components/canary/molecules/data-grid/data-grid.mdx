## Purpose

`DataGrid` is the canonical AG Grid wrapper for the Arda design system. It applies the
`themeQuartz` visual foundation with Arda design tokens (see `ThemedGrid` story) and
handles the boilerplate that every grid needs:

- Column state persistence to `localStorage` via `persistenceKey`
- `SortMenuHeader` for column sort + filter menus
- Loading overlay via `loading` prop
- Custom empty-state slot via `emptyStateComponent`
- Pagination footer via `paginationData` + page callbacks
- Row selection via `enableRowSelection` / `enableMultiRowSelection`
- Cell editing via `enableCellEditing`
- Ref forwarding via `DataGridRef` for imperative access (`exportDataAsCsv`, `getGridApi`)
