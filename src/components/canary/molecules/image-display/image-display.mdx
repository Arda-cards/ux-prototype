# ImageDisplay

Foundational image rendering molecule. Handles all four visual states of an image field: loaded, loading (shimmer), error (broken URL), and no image (null URL).

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/image-display`

---

## Design Decision: No Border

The container uses `bg-muted` as its background. No border is applied &#8212; the muted background
provides sufficient contrast even for white product images.
