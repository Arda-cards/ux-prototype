## Purpose

`ArdaGridAction` renders an icon button with a short label below it, matching the iOS share-sheet
or app-launcher visual pattern. It is the unit cell used in the `ItemDetailsHeader` action grid.

The short label defaults to the first word of `label` if `shortLabel` is not provided. In `loading`
state the icon is replaced with a spinner and the label reads &#8220;Wait&#8230;&#8221;.
