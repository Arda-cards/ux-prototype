# Item Grid Molecules

Shared cell editor and header primitives used inside `ItemGrid`. These molecules are not used
standalone &#8212; they compose into the `ItemGrid` organism via column definitions.

**Layer**: Molecule &#8212; Canary

---

## Playground

`TypeaheadCellEditor` with async supplier lookup. Type to search; press Enter to commit.

<Canvas of={ItemGridMoleculeStories.Playground} />

---

## TypeaheadCellEditor

Async search-as-you-type cell editor for AG Grid. Debounces the `lookup` function and presents
results in a keyboard-navigable dropdown. Supports `allowCreate` to add new values inline.

<Canvas of={ItemGridMoleculeStories.Typeahead} />

<Canvas of={ItemGridMoleculeStories.TypeaheadEmpty} />

---

## SelectCellEditor

Static list dropdown &#8212; the base atom from `@/components/canary/atoms/grid/select`, shown here
for comparison with `TypeaheadCellEditor`.

<Canvas of={ItemGridMoleculeStories.Select} />

## DragHeader

Column header with a drag-grip icon. Used as the `headerComponent` for draggable columns.
