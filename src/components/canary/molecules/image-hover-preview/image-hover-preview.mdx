# ImageHoverPreview

Lightweight hover popover that shows a larger image preview after ~500 ms. Wraps any
trigger element &#8212; typically a thumbnail &#8212; and displays an `ImageDisplay` at ~256&#215;256 inside
a `Popover`.

**Molecule** &#8212; Canary | Import: `@/components/canary/molecules/image-hover-preview`

---

## Playground

Adjust `imageUrl`, `entityTypeDisplayName`, and `propertyDisplayName` in the Controls panel.
Hover the thumbnail to trigger the preview.

<Canvas of={Stories.HoverToPreview} />

---

## Error Suppression

When `imageUrl` is `null`, or when the image enters an error state, the popover is suppressed
entirely &#8212; no preview appears on hover.

<Canvas of={Stories.NoPreviewOnError} />
