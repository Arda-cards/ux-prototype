# Canary Molecules

Molecules compose multiple atoms into functional UI blocks. Each molecule handles a specific interaction pattern &#8212; a toolbar, a data grid, a navigation group &#8212; without containing business logic.

**Import**: `import { DataGrid } from '@arda-cards/design-system/canary'`

---

## Standard Molecules

<table>
  <thead>
    <tr>
      <th>Molecule</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><a href="?path=/docs/components-canary-molecules-actiontoolbar--docs">ActionToolbar</a></td>
      <td>Horizontal bar of icon+label action buttons with an overflow menu for responsive layouts. Used in entity detail headers and selection toolbars.</td>
    </tr>
    <tr>
      <td><a href="?path=/docs/components-canary-molecules-datagrid--docs">DataGrid</a></td>
      <td>Core AG Grid wrapper with Arda theming (<code>themeQuartz</code>), column persistence, sort menu headers, row selection, pagination, and cell editing support. Base for <code>createEntityDataGrid</code>.</td>
    </tr>
    <tr>
      <td><a href="?path=/docs/components-canary-molecules-fieldlist--docs">FieldList</a></td>
      <td>Data-driven label/value list rendering a vertical stack of <code>ReadOnlyField</code> atoms. Used in detail panels and drawers.</td>
    </tr>
    <tr>
      <td><a href="?path=/docs/components-canary-molecules-gridaction--docs">GridAction</a></td>
      <td>Row-level action button group for AG Grid cells. Renders icon buttons with tooltips in a pinned column.</td>
    </tr>
    <tr>
      <td><a href="?path=/docs/components-canary-molecules-overflowtoolbar--docs">OverflowToolbar</a></td>
      <td>Responsive toolbar that measures available width and moves excess items into a dropdown overflow menu. Used in grid selection toolbars and app headers.</td>
    </tr>
  </tbody>
</table>
