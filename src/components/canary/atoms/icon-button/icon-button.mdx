# ArdaIconButton

Icon-only button with accessible label, optional badge count, and tooltip.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/icon-button/icon-button`

---

## Default

<Canvas of={IconButtonStories.Default} />

---

## Purpose

`ArdaIconButton` wraps `ArdaButton` (size `icon`) with:

- A **required `label`** prop that populates `aria-label`.
- An optional **`badgeCount`** for notification-style counts.
- A **tooltip** that surfaces the label on hover/focus.

Used in `ArdaAppHeader` for icon actions (Help, Notifications, User menu).
