# ButtonGroup

Groups related action buttons with connected styling &#8212; collapsed borders and border-radius only on outer edges. Pure CSS layout component with no JavaScript state.

**Atom** &#8212; Canary | Import: `import { ButtonGroup, ButtonGroupSeparator } from '@arda-cards/design-system/canary'`

> Use **ButtonGroup** for buttons that perform actions. Use **ToggleGroup** for buttons that toggle state.

## Dividers

Use `showDivider` to automatically insert separator lines between child buttons. Use `dividerClassName` to control the divider color &#8212; important for solid-background variants where the default `bg-border` would be invisible.

```tsx
{/* Outline buttons — default divider color works */}
<ButtonGroup showDivider>
  <Button variant="outline">Left</Button>
  <Button variant="outline">Right</Button>
</ButtonGroup>

{/* Primary buttons — use a white divider */}
<ButtonGroup showDivider dividerClassName="bg-white/30">
  <Button variant="primary">Save</Button>
  <Button variant="primary">Save &#38; Close</Button>
</ButtonGroup>
```

You can also use `ButtonGroupSeparator` directly for manual placement if you need separators only between specific children.

## Accessibility

- Renders with `role="group"` &#8212; provide `aria-label` to describe the group.
- Focus rings on child buttons layer above siblings via `z-10`.
