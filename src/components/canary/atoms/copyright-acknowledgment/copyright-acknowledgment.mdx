## Purpose

`CopyrightAcknowledgment` is a controlled checkbox atom that ensures users explicitly confirm they own or are licensed to use an image before uploading. When `disabled` is true, the checkbox becomes non-interactive and the container receives `opacity-50`.

The component is intentionally kept stateless &#8212; the parent (e.g., `ImageUploadDialog`) owns the `acknowledged` state and gates the confirm action on it.

## Accessibility

- The checkbox is associated with its label text via `htmlFor` / `id` linkage, so clicking the label text toggles the checkbox.
- The checkbox also carries `aria-label="Copyright acknowledgment"` as a supplementary accessible name.
- When `disabled`, the checkbox is not reachable via keyboard and the visual opacity communicates the inactive state.
