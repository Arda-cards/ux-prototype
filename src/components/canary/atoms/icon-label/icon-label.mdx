# IconLabel

Inline icon + text pair with consistent 16 px icon sizing, 8 px gap, and overflow truncation.

**Atom** &#8212; Canary | Import: `import { IconLabel } from '@arda-cards/design-system/canary'`

## Purpose

`IconLabel` renders a Lucide icon alongside a text label as a single inline `span`. It fixes the icon at 16&#215;16 px via `[&>svg]:size-4`, applies a consistent 8 px gap, and truncates overflow text with an ellipsis when the container is width-constrained.

Use it in navigation items, menu entries, dropdown options, status indicators, and any context where an icon must accompany inline text with predictable alignment.

## Accessibility

- The icon is decorative and carries no independent `aria-label`. The visible `label` text provides the accessible name for the surrounding interactive element (link, button, menu item).
- Do not use `IconLabel` as the sole content of an interactive element without also providing an `aria-label` on that element if the icon is meaningful.
