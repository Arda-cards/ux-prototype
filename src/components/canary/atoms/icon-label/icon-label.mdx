# IconLabel

Inline icon + text pair with consistent sizing and gap.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/icon-label/icon-label`

---

## Default

<Canvas of={IconLabelStories.Default} />

---

## Purpose

`IconLabel` renders a Lucide icon alongside a text label with consistent
sizing and alignment. Used in nav items, status indicators, and anywhere
an icon needs to accompany inline text.
