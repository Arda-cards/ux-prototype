# ArdaButton

Primary action button with multiple variants, sizes, and loading state.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/button/button`

---

## Default

<Canvas of={ButtonStories.Default} />

---

## Purpose

`ArdaButton` is the canonical button atom for all Arda UI. It extends the shadcn
Button primitive with a `loading` prop (spinner overlay) and maps to the Arda
design-system color tokens (primary = orange).
