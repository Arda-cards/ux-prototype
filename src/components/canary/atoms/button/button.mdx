# Button

Canonical action button for all Arda UI. Extends the shadcn Button primitive with a built-in loading state, `motion-reduce` support, and consistent focus rings mapped to Arda design-system color tokens.

**Atom** &#8212; Canary | Import: `import { Button } from '@arda-cards/design-system/canary'`

## Loading State

When `loading` is `true`, a spinner replaces the leading icon slot and the button is disabled. The spinner respects `motion-reduce` (animation stops for users who prefer reduced motion).

<Canvas of={ButtonStories.Loading} />

## Disabled State

Standard HTML `disabled` attribute. Pointer events are suppressed and opacity is reduced.

<Canvas of={ButtonStories.Disabled} />

## Composition: asChild

Use `asChild` to render a different element (e.g., a link) with Button styling via Radix `Slot`:

```tsx
<Button asChild variant="ghost">
  <a href="/items">View all items</a>
</Button>
```

## Accessibility

- Focus ring: 2px `ring` token, visible on keyboard focus via `focus-visible`.
- Destructive variant uses `ring-destructive` for its focus indicator.
- `motion-reduce`: loading spinner animation stops for users who prefer reduced motion.
- Icon-only buttons require `aria-label`.
