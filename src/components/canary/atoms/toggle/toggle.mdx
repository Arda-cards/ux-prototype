# Toggle

A two-state button that can be on or off. Wraps Radix Toggle with Arda conventions &#8212; CVA variants aligned to the Button size scale, `motion-reduce` support, and consistent focus rings.

**Atom** &#8212; Canary | Import: `import { Toggle } from '@arda-cards/design-system/canary'`

## Accessibility

- Uses Radix Toggle primitive &#8212; `aria-pressed` is managed automatically.
- Keyboard: `Space` and `Enter` toggle the pressed state.
- Focus ring matches Button's 2px `ring` token.
