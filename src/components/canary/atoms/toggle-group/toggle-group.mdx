# ToggleGroup

A set of two-state toggles with single or multiple selection. Wraps Radix ToggleGroup with Arda conventions &#8212; shared CVA variants passed via context, discriminated union types for single vs. multiple mode.

**Atom** &#8212; Canary | Import: `import { ToggleGroup, ToggleGroupItem } from '@arda-cards/design-system/canary'`

## Accessibility

- Single mode: items have `role="radio"` with `aria-checked`.
- Multiple mode: items behave as independent toggles.
- Keyboard: Arrow keys navigate between items, `Space`/`Enter` toggle selection.
- Variant and size are passed via context &#8212; items are visually consistent automatically.
