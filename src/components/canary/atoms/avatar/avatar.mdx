# ArdaAvatar

User avatar atom with image, initials fallback, and size variants.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/avatar/avatar`

---

## Purpose

`ArdaAvatar` wraps the Radix Avatar primitive. When an image URL is provided it
renders the image; when the image fails to load (or no URL is given) it falls
back to initials computed from the user's name. Used in `SidebarUserMenu` and
profile display contexts.
