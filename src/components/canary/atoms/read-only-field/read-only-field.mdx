# ReadOnlyField

Label + value display atom for detail views and read-only forms.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/read-only-field/read-only-field`

---

## Default

<Canvas of={ReadOnlyFieldStories.Default} />

---

## Purpose

`ReadOnlyField` displays a `label` (muted, small) above a `value` (primary text).
Used extensively in `ArdaFieldList` and supplier/item detail drawers to render
named data fields in a consistent visual format.

Replaces the deprecated `ArdaDetailField` &#8212; do not use `ArdaDetailField` in
new code.
