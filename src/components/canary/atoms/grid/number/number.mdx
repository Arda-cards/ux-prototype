# Number Cell

Display and editor for numeric grid fields. Right-aligned with tabular-nums font feature for clean column alignment. Supports configurable decimal precision.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)
