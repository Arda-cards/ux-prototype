# Action Cell Renderer

Vertical-ellipsis button (&#8942;) that opens a portal dropdown menu of row-level actions. Designed for use as an AG Grid cell renderer in a pinned-right column.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)

## Design notes

- The dropdown menu renders via `createPortal` to `document.body`, so it escapes AG Grid&#8217;s `overflow: hidden` cell container.
- Menu position is computed from the trigger button&#8217;s `getBoundingClientRect`.
- Keyboard: `Escape` closes the menu; `Enter`/`Space` open it on the trigger.
- Clicking outside the menu closes it.
