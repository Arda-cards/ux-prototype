# Select Cell

Display and editor for enum-backed grid fields. The display looks up a human-readable label from the options map; the editor renders a keyboard-navigable ARIA listbox popup.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)

## Accessibility

- The popup editor renders a `role="listbox"` with `role="option"` children.
- The active/highlighted option is tracked via `aria-selected`.
- Keyboard bindings: Arrow Down/Up to navigate, Enter to commit, Escape to cancel.
