# Boolean Cell

Display and editor for true/false grid fields. Supports two visual formats: a checkbox icon or a &#8220;Yes&#8221;/&#8220;No&#8221; text label.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)
