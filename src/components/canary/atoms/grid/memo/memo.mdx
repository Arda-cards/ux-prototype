# Memo Cell

Display and editor for multi-line text grid fields. The display truncates at a configurable character limit and shows the full text on hover. Provides two editor variants: an inline `textarea` and a button-cell that opens a modal dialog.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)

### MemoButtonCell

Use `MemoButtonCell` for a read-only display cell with a pencil icon that opens a modal dialog editor. This is the preferred pattern for long-form text that shouldn&#8217;t be edited inline.
