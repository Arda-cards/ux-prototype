# Date Cell

Display and editor for ISO date string grid fields. Renders dates in locale-aware format (MM/DD/YYYY by default) and provides a native date-picker editor.

**Grid Atom** &#8212; see [Grid Guide](?path=/docs/components-canary-atoms-grid-guide--docs)

## Props &#8212; Editor Factory (`createDateCellEditor`)

The factory accepts no required configuration. The resulting editor renders a native `&#60;input type="date"&#62;` inside the AG Grid cell.
