# Canary Atoms

Atoms are the smallest functional UI building blocks in the Arda Canary component system. Each atom encapsulates a single, focused concern &#8212; a button, a badge, a search field &#8212; and carries no business logic. Molecules and organisms are built by composing atoms together.

All atoms are exported from the canary barrel:

```tsx
import { Avatar, Badge, Button, Card, Dialog, Drawer, SearchInput } from '@arda-cards/design-system/canary';
```

## Grid Cell Atoms
