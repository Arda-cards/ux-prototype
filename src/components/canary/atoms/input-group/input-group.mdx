# InputGroup

Labelled input field with optional leading icon and helper/error text.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/input-group/input-group`

---

## Purpose

`InputGroup` composes a shadcn `Label`, a shadcn `Input`, and optional helper or
error text into a single accessible form control. Use it wherever a labelled input
is needed in forms and dialogs.
