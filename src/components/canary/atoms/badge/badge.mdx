# Badge

Status badge atom with count capping and dot-indicator support.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/badge/badge`

---

## Default

<Canvas of={BadgeStories.Default} />

---

## Purpose

`Badge` wraps the shadcn `Badge` primitive with two Arda-specific features:

1. **Count mode** &#8212; pass `count={n}` to render a live count badge. Capped at `max` (default 99). Renders as `99+` when exceeded.
2. **Dot mode** &#8212; pass `count={true}` (boolean) to render a dot indicator regardless of count. Used in collapsed sidebar nav items.
