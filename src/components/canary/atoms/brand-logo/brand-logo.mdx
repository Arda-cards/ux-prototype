# BrandLogo

Arda brand identity atoms in four color variants.

**Atom** &#8212; Canary | Import: `import { BrandLogo, BrandIcon } from '@arda-cards/design-system/canary'`

## Purpose

Two separate components serve different contexts:

- **`BrandLogo`** is the full Arda wordmark (55&#215;30 px). Use it in expanded sidebars, app headers, and login pages.
- **`BrandIcon`** is the compact square mark (30&#215;30 px). Use it in collapsed sidebars, favicons, and mobile headers where horizontal space is constrained.

Both components render an `<img>` element with a fixed `alt="Arda"` attribute and accept the same `variant` and `className` props.

## Accessibility

- Both components render `alt="Arda"` on the underlying `<img>` element, satisfying alt-text requirements.
- If the logo is purely decorative (adjacent branded text already identifies the product), pass `className` with `aria-hidden` applied directly, or wrap in a container with `aria-hidden="true"`.
