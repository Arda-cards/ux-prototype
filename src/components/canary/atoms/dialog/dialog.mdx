# ArdaDialog

Modal dialog atom for confirmations, forms, and alerts.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/dialog/dialog`

---

## Purpose

`ArdaDialog` wraps the Radix Dialog primitive with Arda styling (rounded corners,
backdrop blur, design-system spacing). Used for confirmation dialogs (delete, discard),
quick-add forms, and error alerts. For slide-over panels use `Drawer` instead.
