# SplitButton

A two-part button: primary action on the left, dropdown trigger (chevron) on the right. Composes Button, ButtonGroup, and DropdownMenu. The left button fires `onClick`; the right chevron opens a dropdown menu with alternate actions.

**Atom** &#8212; Canary | Import: `import { SplitButton } from '@arda-cards/design-system/canary'`

## Variants

All five Button color variants are supported. The separator between the action and chevron is styled per variant &#8212; translucent white for solid backgrounds, `bg-border` for outline/ghost.

## Loading

The `loading` prop disables both the action button and the chevron. When loading is a string, it replaces the action button's children text.

## Menu Content

The `menuContent` prop accepts any `DropdownMenu` primitives &#8212; compose your menu items freely:

```tsx
<SplitButton
  menuContent={
    <>
      <DropdownMenuItem>Save as draft</DropdownMenuItem>
      <DropdownMenuItem>Save and close</DropdownMenuItem>
      <DropdownMenuSeparator />
      <DropdownMenuItem variant="destructive">Discard</DropdownMenuItem>
    </>
  }
>
  Save
</SplitButton>
```

## Accessibility

- The chevron trigger has `aria-haspopup="menu"` and `aria-expanded` managed by Radix.
- Provide `menuLabel` for a descriptive `aria-label` on the chevron (defaults to "More options").
- Keyboard: `Tab` moves between the two buttons; `Enter`/`Space` activates; `ArrowDown` on the trigger opens the menu.
