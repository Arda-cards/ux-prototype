# ArdaSearchInput

Search field with a leading search icon.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/search-input/search-input`

---

## Default

<Canvas of={SearchInputStories.Default} />

---

## Purpose

`ArdaSearchInput` wraps the shadcn `Input` primitive with a positioned `Search` icon
from Lucide. Used in the entity-data-grid search bar and wherever a standalone search
input is needed. The component is controlled: pass `value` and `onChange` to wire it
to state.
