## Purpose

`ArdaSearchInput` wraps the shadcn `Input` primitive with a positioned Lucide `Search` icon pinned to the left edge. The input is controlled: pass `value` and `onChange` to wire it to local or external state. Used in entity-data-grid toolbars and any standalone search context.

## Accessibility

- The underlying `<input>` receives `type="search"` and an `aria-label` equal to the `placeholder` value. Pass a descriptive placeholder to ensure an informative label.
- The search icon is `aria-hidden="true"` and `pointer-events-none` so it does not interfere with interaction or assistive technology.
