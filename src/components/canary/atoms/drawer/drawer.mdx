# Drawer

Slide-over drawer atom for detail panels and contextual editing.

**Atom** &#8212; Canary | Import: `@/components/canary/atoms/drawer/drawer`

---

## Default

<Canvas of={DrawerStories.Default} />

---

## Purpose

`Drawer` slides in from the right side of the viewport. Use it for detail
panels, edit forms, and contextual actions that need more space than a dialog.
For simple confirmations use `ArdaDialog` instead.

The `ItemDetails` organism builds on top of `Drawer` &#8212; prefer using
`ItemDetails` for item-specific detail panels.
