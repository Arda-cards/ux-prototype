# ItemDetails

Item detail slide-over panel with tabbed navigation, field list, card preview, and action toolbar.

**Organism** &#8212; Canary | Import: `@/components/canary/organisms/item-details/item-details`

## Purpose

`ItemDetails` is a compound organism built from `Drawer`, `ItemDetailsHeader`,
`FieldList`, `ItemDetailsCardPreview`, and `ActionToolbar`. It manages internal
tab state and card navigation.

Open the panel by setting `open={true}` and wiring `onOpenChange` to your state. Pass
`fields` for the details tab and `renderCard` + `cardCount` for the cards tab.

## Tabs

The default configuration includes two tabs:

<table>
  <thead>
    <tr>

    <tr>

      <td>Item details</td>
      <td>Read-only field list from the <code>fields</code> prop.</td>
    </tr>
    <tr>
      <td><code>cards</code></td>
      <td>Cards</td>
      <td>Card carousel rendered via <code>renderCard</code> / <code>cardCount</code>.</td>
    </tr>
  </tbody>
</table>

Override tabs entirely by passing a custom `tabs` array. Use `renderCardsTab` for fully
custom cards tab content.
