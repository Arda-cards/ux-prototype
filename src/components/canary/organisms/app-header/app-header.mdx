### Actions

Actions use a data-driven array pattern:

- **`buttonActions`** &#8212; labeled buttons with icons (e.g., "Scan"). Rendered before icon actions.
- **`actions`** &#8212; icon-only buttons with optional badge counts (e.g., Help, Notifications). Set `visible: false` to conditionally hide.

### Leading content

Use the `leading` prop to render a sidebar trigger, breadcrumbs, or other content on the far left.
