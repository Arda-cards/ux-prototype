# Entity Data Grid

General-purpose data grid factory for building typed, feature-rich entity list views. All entity grids in Arda (ItemGrid, supplier grid, etc.) are built on this foundation.

**Organism** &#8212; Canary | Factory: `createEntityDataGrid<T>(config)`

## Factory Pattern

```tsx
const { Component: ItemGrid } = createEntityDataGrid<Item>({
  displayName: 'ItemGrid',
  persistenceKeyPrefix: 'item-grid',
  columnDefs: itemGridColumnDefs,
  defaultColDef: itemGridDefaultColDef,
  getEntityId: (item) => item.entityId,
  searchConfig: { fields: ['name', 'internalSKU'] },
  enableDragToScroll: true,
});
```

The factory returns `{ Component }` &#8212; a fully configured React component that accepts runtime props (`data`, `loading`, `onRowPublish`, etc.).
