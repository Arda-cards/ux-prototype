# Canary Organisms

Organisms are the largest reusable UI compositions in the canary component system. Each organism assembles multiple molecules and atoms into a complete, self-contained UI section &#8212; a sidebar, an app header, a data grid page, or an entity detail panel.

**Import**: `import { Sidebar, AppHeader, ItemGrid } from '@arda-cards/design-system/canary'`

---
