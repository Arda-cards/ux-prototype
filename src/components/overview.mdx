# Components Section Overview

The component library is organized into three collections by maturity.

## Stable

`Components/Stable` is the root export path for production-ready components.

## Canary

`Components/Canary` is a sub-package for components under active extraction and refinement. Promotion to Stable requires an explicit move and import change.

## Extras

`Components/Extras` contains early experiments and reference implementations kept as examples. These are not exported in the library and will be removed as real components replace them.
