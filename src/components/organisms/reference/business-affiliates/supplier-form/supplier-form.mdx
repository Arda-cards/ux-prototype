# ArdaSupplierForm

Shared form component for creating and editing business affiliates (suppliers, customers, carriers, etc.).

## Two Presentation Modes

**Single-scroll** &#8212; all three sections render vertically with collapsible panels for Contact and Address/Legal.
Used inside the `ArdaSupplierDrawer` for add/edit operations.

**Stepped** &#8212; only the section matching `currentStep` renders.
Used inside the Create Supplier wizard where each section maps to a wizard step.
