# Archive

This section contains historical application mocks from earlier design stages. These stories were created during initial prototyping and are preserved here for reference. They are not actively maintained and may not reflect the current state of the application.

## Contents

- **Applications/Design** &#8212; Hand-built page-level design mocks created before the Dev Witness integration existed.
- **Applications/Development** &#8212; Development-stage mocks at intermediate fidelity.
- **Applications/Production** &#8212; Production-parity mocks (pre-vendored approach).

## Why Archive?

The **Dev Witness** section now provides a more accurate representation of the production application by rendering actual vendored source code against MSW mocks. The archived application mocks served as stepping stones toward that approach and remain available for historical context.
