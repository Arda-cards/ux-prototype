# Arda UX Prototypes

Wireframe prototype gallery for Arda frontend components. Generated by AI agents, reviewed by humans.

## Quick Start

```bash
make install   # Install dependencies
make dev       # Start dev server at http://localhost:3000
```

Run `make help` to see all available targets.

## Creating a New Prototype

1. **Copy the sample folder:**
   ```bash
   cp -r src/app/prototypes/sample src/app/prototypes/my-feature
   ```

2. **Edit `src/app/prototypes/my-feature/page.tsx`** with your wireframe components.

3. **Register it in the gallery** by adding an entry to the `prototypes` array in `src/app/page.tsx`.

4. Run `make dev` and browse to `http://localhost:3000/prototypes/my-feature`.

## Wireframe Mode

Click the **Wireframe** button in the header to toggle a sketch aesthetic:
- Grayscale palette
- Hand-drawn font (*Indie Flower*)
- Dashed borders

This prevents reviewers from being distracted by visual polish during structural review.

## Technology Stack

Aligned with `arda-frontend-app`:

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript** (strict mode)
- **Tailwind CSS v4** (`@tailwindcss/postcss`)
- **shadcn/ui** (new-york style, lucide icons)
